# Economic Impact of War – Data Analysis & Power BI Dashboard

## Overview

This project analyzes the economic and social impact of armed conflicts using a dataset containing economic indicators such as GDP change, inflation, unemployment, poverty levels, currency devaluation, and reconstruction costs.

The analysis was performed using **Python for exploratory data analysis (EDA)** and **Power BI for visualization**.

---

## Tools Used

* Python (Pandas, NumPy, Matplotlib)
* Jupyter Notebook
* Power BI

---

## Key Insights

* Civil wars cause the **largest GDP decline**.
* The **Middle East region experiences the highest unemployment increase**.
* Informal economies expand significantly during wartime.
* Reconstruction costs often exceed initial war costs.

---

## Dashboard Features

The Power BI dashboard includes:

* GDP Change by Conflict Type
* Currency Devaluation vs Black Market Activity
* Poverty Increase by Region
* Unemployment Increase by Region
* War Cost vs Reconstruction Cost
* Informal Economy Growth
* Conflict Trends Over Time
* Most Affected Economic Sectors
* Interactive filters for Region, Conflict Type, and Status
